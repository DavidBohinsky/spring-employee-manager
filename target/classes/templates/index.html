This page serves as an administrative interface for managing employees within the company’s internal database.

– View a list of all employees with details such as first name, last name, email, date of birth, company, bio, and photo.
– Add new employees or companies using form-based input.
– Edit existing employee records (via the Edit button).
– Delete entries (via the Delete button).
– Manage data in a modern, user-friendly table layout.
